# DAoC Starter - STILL WORK IN PROGRESS!!!
a Dark Age of Camelot client starter

## based on:

+ NodeJS
+ Electron
+ Express
+ Datatables.net
+ jQuery
+ tingodb (mongodb)
+ Bootstrap
+ scrollreveal

## developer's installation:

1. git clone
2. npm i

## developer's debug start:

1. npm start
